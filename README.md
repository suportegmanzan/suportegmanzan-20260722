# Sistema de Gestão de Colaboradores e Unidades

- **Backend**: C# (.NET 8, ASP.NET Core Web API)
- **Front**: Angular (standalone components, Angular 17)
- **Banco**: PostgreSQL, rodando em Docker
- **Autenticação**: Bearer token (JWT)
- **Arquitetura**: MVC (Models / Controllers / Services) - herança `EntidadeBase`
